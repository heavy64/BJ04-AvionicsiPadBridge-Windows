# Avionics Buddy Bridge for Windows

Avionics Buddy Bridge is the Windows companion app for Avionics Buddy.

It connects Microsoft Flight Simulator on your Windows PC to the Avionics Buddy iPhone/iPad app over your local network.

## What it does

- Connects to Microsoft Flight Simulator 2020 or 2024
- Reads live simulator telemetry through SimConnect
- Serves telemetry to Avionics Buddy over the local network
- Shows a QR code and pairing link for easy setup
- Works locally — no account, no cloud telemetry

## Requirements

- Windows 10 or later
- Microsoft Flight Simulator 2020 or 2024
- iPhone or iPad with Avionics Buddy installed
- PC and iPhone/iPad connected to the same local network
- Local network access allowed through Windows Firewall

## Install

1. Download the latest ZIP from the Releases page.
2. Extract the ZIP.
3. Open the extracted folder.
4. Run `AvionicsiPadBridge.exe`.
5. If Windows asks for firewall permission, allow local network access.
6. Open Avionics Buddy on iPhone or iPad.
7. Scan the QR code or paste the pairing link.

## Privacy

Avionics Buddy Bridge runs locally on your Windows PC.

It does not upload simulator telemetry to the cloud. Flight data is sent only across your local network to paired Avionics Buddy devices.
